## Teslo Api
1. Clone repo
```
git clone https://github.com/repo
```
2. Run command
```
yarn install
```
3. Clone file ```.env.template``` and rename ```.evn```
4. Update environment
5. Up db
```
docker-compose up -d
```
6. Run dev mode
```
yarn start:dev
```
7. Run seed data
```
http://localhost:3000/api/seed
```
